Deploying Flask with Heroku

Go to Python directory, cd into scripts and pip install vituralenv

cd into directory you want to create project

to create virtualenv type virtualenv *name*

to activate virtualenv source *name*/bin/activate

pip install flask

pip install gunicorn - better webserver

create Procfile - web: gunicorn *name of python file*:app

heroku create

git init, git add ., git commit, git push heroku master

pip freeze > requirements.txt
