Connectives(Operators): Given two propositions, we can connect them together: And $(\wedge)$,Or $(\lor)$, Not $(\neg)$, Implication $(\Rightarrow)$

$P \Rightarrow$ Q$ If P is true then Q is True, if P is false, this is vacously true - which means the implication is automatically true if P is false.

Contrapositive: $P \Rightarrow Q \equiv \neg Q \Rightarrow \neg P$

$P \Rightarrow Q \equiv \neg P \lor Q$

Converse: $Q \Rightarrow P$

$ P \iff Q$

$ P \equiv Q $ For definitions
