Deploying Flask with Heroku

This is generally done in a virtual environment

cd into Python directory, cd into scripts and pip install vituralenv

cd into directory you want to create project

to create virtualenv: 

$ virtualenv *name* // generally env

to activate virtualenv source: 

$ source env/bin/activate

pip install flask

pip install gunicorn, which is A WSGI HTTP server that can handle requests faster than native Python

create Procfile - web: gunicorn *name of python file*:app 

$ heroku create

$ git init

$ git add .

$ git commit -m "commit"

$ git push heroku master 

$ pip freeze > requirements.txt

Value Investing

"Rule No. 1: Never lose money. Rule No. 2: Never forget rule No.1" ~ Warren Buffet

How do you not lose money?

Invest in mostly safe stocks (80%) that have the most value for the least price

Value is considered as Net Earnings(Retained Earnings) + Dividends (Cash Distribution by Company to shareholders)

(Start-ups and other high-growth companies especially in the technology or biotechnology sectors rarely offer dividends because all of their profits are reinvested to help sustain higher-than-average growth and expansion)

Price of a stock is equal to the Book ratio which is the Share price/ Book value per share)

Book value per share = (assets - liabilities of a company) /total shares ( Gives as quantity for every share, company has X amount of assets)

If ratio comes out to >10, You should buy

Want Companies that have high ROCE (Return on capital employed) and low debt ( Debt/equity ratio - calculated by dividing a company’s total liabilities by its stockholders' equity.

The D/E ratio indicates how much debt a company is using to finance its assets relative to the amount of value represented in shareholders’ equity.)

Common safe stocks : Coca - Cola, Banks, ETFs (Tech is very risky)
